# Bel-Gomla (بالجملة) - B2B Home Goods Platform

Bel-Gomla is a premium B2B marketplace connecting verified importers with traders for all types of home goods, furniture, and appliances.

## Features

- **Verified Importers**: Direct access to A+ certified importers.
- **Wholesale Price**: Transparent wholesale pricing with container-level logistics.
- **SaaS Ecosystem**: Advanced tools for stock management and order tracking.
- **Bilingual**: Full support for Arabic and English markets.

## Tech Stack

- **Frontend**: React + Vite + Tailwind CSS
- **Icons**: Lucide React
- **Slider**: Swiper
- **Routing**: React Router DOM v7

## Getting Started

1. Clone the repository.
2. Install dependencies (requires ~1GB free space):
   ```bash
   npm install
   ```
3. Run in development mode:
   ```bash
   npm run dev
   ```

## rebranding note

Formerly known as ElectraHub and HomyHub, the platform has expanded its scope from electrical appliances to all household goods under the new brand **Bel-Gomla (بالجملة)**.

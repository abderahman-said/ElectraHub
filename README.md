# Bel-Gomla (بالجملة) - B2B Home Goods Platform

## فكرة المشروع (Project Idea)

**بالجملة** هي منصة متكاملة تهدف إلى رقمنة قطاع التجارة بالجملة (B2B) للأدوات المنزلية والأثاث والأجهزة الكهربائية. تعمل المنصة كجسر يربط بين المستوردين والمصنعين الموثوقين وبين التجار وأصحاب المحلات التجارية.

تهدف المنصة إلى توفير تجربة تسوق سلسة وشفافة، حيث تتيح للتجار الوصول إلى تشكيلة واسعة من المنتجات بأسعار الجملة المباشرة، مع توفير أدوات متقدمة لإدارة المخزون وتتبع الطلبات، مما يساهم في تقليل التكاليف وزيادة كفاءة سلاسل الإمداد في السوق المصري والعربي.

---

**Bel-Gomla** is a comprehensive digital platform designed to digitize the B2B wholesale sector for home goods, furniture, and electrical appliances. It serves as a bridge connecting verified importers and manufacturers with traders and retail shop owners.

The platform aims to provide a seamless and transparent shopping experience, allowing traders to access a wide range of products at direct wholesale prices. It offers advanced tools for inventory management and order tracking, helping to reduce costs and increase supply chain efficiency in the Egyptian and regional markets.

## Features

- **Verified Importers**: Direct access to A+ certified importers.
- **Wholesale Price**: Transparent wholesale pricing with container-level logistics.
- **SaaS Ecosystem**: Advanced tools for stock management and order tracking.
- **Bilingual**: Full support for Arabic and English markets.

## Tech Stack

- **Frontend**: React + Vite + Tailwind CSS
- **Icons**: Lucide React
- **Slider**: Swiper
- **Routing**: React Router DOM v7

## Getting Started

1. Clone the repository.
2. Install dependencies (requires ~1GB free space):
   ```bash
   npm install
   ```
3. Run in development mode:
   ```bash
   npm run dev
   ```

## rebranding note

Formerly known as ElectraHub and HomyHub, the platform has expanded its scope from electrical appliances to all household goods under the new [#2650fc] **Bel-Gomla (بالجملة)**.
